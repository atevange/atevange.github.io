Google Classroom

  • Intro: The main objective of this project is to evaluate and improve the website "Google Classroom" using a heuristic evaluation
  • Analysis: Heuristic evaluation helped identify minor issues in Google Classroom which generated the task given to the participants
  • Prototype & Design: The prototype was redesigned to look like the original website but less cluttered and more accessible for the user
  • A&B Testing: Original and prototype website's were both tested by particpants. Recorded particpants interaction with websites and had a pre/post survey
